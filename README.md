# streamlit
# 🚗 2024년 교통사고 발생률 데이터 분석 프로젝트

## 📌 프로젝트 설명
본 프로젝트는 2024년 국내 교통사고 발생 데이터를 기반으로 사고 발생률을 분석하고, 주요 요인을 파악하여 교통 안전 개선에 기여할 수 있는 인사이트를 도출하는 것을 목표로 합니다.  
데이터 전처리, 시각화, 그리고 간단한 예측 모델을 통해 사고 발생 패턴을 이해하고 정책적 시사점을 제시합니다.

---

## 🎯 주제
- 2024년 교통사고 발생률 분석

---

## 💡 주제 선택 이유
- 교통사고는 국민 안전과 직결되는 중요한 사회 문제입니다.  
- 2024년 데이터를 활용하여 최신 경향을 반영할 수 있습니다.  
- 사고 발생률을 분석함으로써 예방 정책 및 안전 캠페인에 활용할 수 있습니다.  

---

## 📊 데이터 분석 내용
1. **데이터 수집 및 전처리**
   - 2024년 교통사고 공개 데이터셋 활용
   - 결측치 처리 및 변수 정리
2. **탐색적 데이터 분석 (EDA)**
   - 월별/지역별 사고 발생률 시각화
   - 사고 유형별 분포 분석
3. **모델링**
   - 사고 발생률 예측을 위한 회귀/분류 모델 적용
4. **결과 및 인사이트**
   - 특정 시간대와 지역에서 사고가 집중됨을 확인
   - 사고 예방을 위한 정책적 제언
